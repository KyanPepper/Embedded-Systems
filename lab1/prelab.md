What flag for ls displays all files, even hidden ones?

**Answer:** `-a` (e.g., `ls -a`)

---

How do you move to the parent directory of the current one?

**Answer:** `cd ..`

---

What is the difference between the mv and cp commands?

**Answer:** `mv` moves or renames a file (the original is removed), while `cp` copies a file (the original remains in place).

---

What does the -h flag of ls do? Hint: use man to find out.

**Answer:** `-h` displays file sizes in human-readable format (e.g., 1K, 234M, 2G) instead of bytes. Used with `-l` or `-s`.

---

In a git repository, what command displays whether there are any local changes and what they are?

**Answer:** `git status`

---

In a git repository, what does git pull do?

**Answer:** `git pull` fetches changes from the remote repository and merges them into your current local branch. It's essentially `git fetch` followed by `git merge`.
